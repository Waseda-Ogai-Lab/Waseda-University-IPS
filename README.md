## 看病：

在日本大家得病了不是首先去大医院（附近有产业医科大医院，算是不错的综合医院），而是先去专门的小诊所看，如果小诊所的医生觉得实在治不了，会给你推荐信去大医院看；如果没有推荐信的情况下直接去产业医科大医院，第一次需要缴纳5000日元的费用（算是挂号费之类的）。ps：产医科大的医生可以用英文交流。面诊完后，医院不会直接给你开药，而是给你处方去医院附近的药店配药。

医院的诊疗费和买药的药费都可以用国民健康保险报销70%（5000日元不在报销范围中），报销完后，还可以拿着你缴费的单子去office通过校友健康互助报销剩余费用（一个机构你只需要出1000日元。比如医院看病花了10000，买药花了5000，医保报销之后就是医院出3000，买药出1500，然后去office报销，会返还给你2000+500。pps：需要等三四个月才能到账。）

有智齿的同学可以享受福利啦，学校报销拔一颗只需要自己花1000。

## 日语

建议来之前至少学个五十音不然字都不会拼 能达到N3以上最好

（想要打工至少N2水平或者具备口语基本交流能力，或者打不怎么需要日语的苦力）

开学会有个日语课报名，财团资助部分学费，一学期自己交12000日元，出勤不到70%需要补交财团资助的12000，有个简单的分班考试，看你日语水平。然后一周上两次课。教材是《大家的日本语》的教材和配套的翻译文法解读，大家可以先在国内买了带来，这边&&文具店大概卖2000日元，上课进度较慢。一学期初级班才上了课本上的10课。自己交8000元。日本《大家的日语》书2700 《翻译与文法讲解》2000 如果要上的话还是自己买了背过来

## 交通：

### 自行车：

自行车与机动车同等对待，出事按车祸处理。

骑自行车不可以带人，经过路口需要减速查看，转弯过马路需要下车推行。

自行车要配车灯，买的时候要办一个防犯登记（几百块登记费），然后就有个贴纸贴车上，被盗后警察凭号码找到主人。

nafco的好贵。早来可以收二手。

### 乘车：

Google maps好用，Google maps里面没有小春门口的公交信息，搜索去折尾的话会先让走到本城西团地，请使用yahoo换乘案内。

交通费巨贵，基本上公共交通相当于国内打车，打车相当于国内专车。

公交车与JR起步价190日元，每坐一站+40日元；taxi起步价700日元，每一公里+约300日元（堵车的话还有低速行驶费之类的，深夜还要加20%车费）。

也可以坐电车（1200+，普通/快速 较慢，班次多时间任性，也有高速电车可选）

## 打工：

不建议不缺钱的同学去打工，想体验生活不如出去旅游。

不建议第一学期出去打工，课程成绩还是最重要的，另外日语学好了可以找好一些的工作。

在入境的时候在海关说 part-time job（アルバイト），或者来之后，去小仓的福冈县入国管理局北九州出张所（小仓城附近）办资格外活动证明（不是就劳证明）

过去办理的时候一定要带护照和在留卡！

没有收费，就填一个表。

大部分需要一定的日语水平。
